======================================================
THƯ MỤC DÀNH RIÊNG CHO MÀNG BỌC NILON BẢO VỆ RUỘNG (PLASTIC MULCH)
======================================================

1. Thả file Model 3D màng nilon mới (.fbx, .obj, .glb) vào thư mục này.
2. Thả hình ảnh Texture (.png, .jpg) màng nilon vào đây.
3. Tạo và lưu Prefab hoàn chỉnh 'PlasticMulch.prefab' tại thư mục này.
