# Project Overview

## Project Name

Đất Cày Lên Sỏi Đá

## Genre

3D Farming Simulation, Village Survival, Community Management

## Engine

Unity 6

## Goal

Người chơi vào vai Thành:
* Khai hoang đất đá.
* Trồng trọt.
* Giúp đỡ dân làng.
* Chống thiên tai.
* Xây dựng cộng đồng.

## Current Status

Prototype / Pre-alpha. Basic character controls, farming, weather, and inventory systems are implemented and functional.

## Core Features

* **3D Movement & Animations**: Walk/run, action locks for tilling/watering/harvesting.
* **Farming**: Tilling (clearing rocks), planting seeds, watering, and harvesting.
* **Crop Visuals**: Stage growth updates (seedling, growing, ready) with fallback 3D primitive geometry.
* **Community System**: Affection points, vần công credits.
* **Weather System**: Seasons (Lập Nghiệp, Gió Lào, Mưa Bão, Phù Sa) affecting temperature, humidity, wind, and water level.
* **Inventory & UI**: Canvas HUD displays survival stats, time, weather overlays, and alerts via HUD toast notifications.
